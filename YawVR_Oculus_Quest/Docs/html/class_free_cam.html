Detailed Description

A simple free camera to be added to a Unity game object.

Keys: wasd / arrows - movement q/e - up/down (local space) r/f - up/down (world space) pageup/pagedown - up/down (world space) hold shift - enable fast movement mode right mouse - enable free look mouse - free look / rotation
